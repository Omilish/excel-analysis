## Аналитика продаж и конкурентной позиции (Excel)

### Стэк: Excel(функции, ВПР, сводные таблицы, визуализация)

### Задачи:
- Проанализировала динамику продаж компании и рынка за 3 года.
- Проверила гипотезы, построила визуализации по каналам и сегментам.

### Методы: визуализация, расчет статистик и метрик

### Результат:
- бренд остаётся лидером рынка;
- выявлено снижение темпов роста;
-  основной объём продаж формируют супермаркеты.
